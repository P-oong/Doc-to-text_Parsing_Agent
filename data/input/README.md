# 입력 파일 디렉토리

여기에 처리할 PDF 또는 HWP 파일을 넣어주세요.

## 지원 형식

- `.pdf` - PDF 문서
- `.hwp` - 한글 문서 (변환 필요)

## 사용 방법

1. 파일을 이 디렉토리에 복사
2. 메인 스크립트 실행:
   ```bash
   python main.py --input-dir data/input/
   ```

## 예시

```
data/input/
  ├── 재무보고서_2023.pdf
  ├── 카카오뱅크_20240508_한화투자증권.pdf
  └── KRX_증권과투자따라잡기.pdf
```

## 주의사항

- 파일명에 한글 사용 가능
- 대용량 파일의 경우 처리 시간이 길어질 수 있습니다
- 암호화된 PDF는 지원하지 않습니다
